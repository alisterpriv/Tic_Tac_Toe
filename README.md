# Tic Tac Toe

**Tic Tac Toe** is a classic 2-player game built using HTML, CSS, and JavaScript. This project demonstrates essential web development skills, such as DOM manipulation, event handling, and responsive design.

## Features

- **Interactive 3x3 Grid**: Play X vs O games in your browser.
- **Turn Indicator**: Shows which player's turn is next.
- **Win/Draw Detection**: Highlights the winner and announces a draw if the grid is full.
- **Restart Option**: Quickly reset the board to start a new game.
- **Responsive UI**: Works smoothly on both desktop and mobile devices.
- **Simple Design**: Clean CSS styling for easy readability and fun gameplay.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/alisterpriv/Tic_Tac_Toe.git
   ```
2. **Open the game:**
   - Navigate to the project folder.
   - Double-click `index.html` to launch the game in your browser.

## How to Play

- Player 1 is **X**, Player 2 is **O**.
- Alternate turns by clicking on an empty square.
- The first player to align three marks horizontally, vertically, or diagonally wins.
- If the grid is filled and there’s no winner, it’s a draw!

## Files

- `index.html` — Main HTML file
- `style.css` — Custom CSS styling
- `script.js` — Game logic with JavaScript
